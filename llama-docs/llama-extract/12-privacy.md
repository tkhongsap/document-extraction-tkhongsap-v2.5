## Key Privacy Statement

According to the LlamaExtract documentation, the service prioritizes user data protection with this assurance: "Your data is kept private and is used only to return your results, never for model training."

## What This Means

The documentation emphasizes that privacy considerations are significant when working with sensitive documents. The platform commits to:

- **Data confidentiality**: Information processed through LlamaExtract remains private
- **Limited data use**: Data serves exclusively for generating extraction results
- **No model training**: User data will not be utilized to improve or train underlying models

This privacy approach is particularly relevant for organizations handling confidential business information, financial records, or other sensitive documentation that requires strict data protection protocols.
