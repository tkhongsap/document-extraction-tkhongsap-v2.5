# LlamaParse Supported Document Types

**Source:** https://developers.llamaindex.ai/python/cloud/llamaparse/features/supported_document_types/

LlamaParse accepts a diverse range of file formats across multiple categories:

## Base Types
PDF files form the foundation of supported formats.

## Documents and Presentations
The platform handles traditional office documents including ".doc, .docx, .ppt, .pptx" and numerous legacy formats. It also supports ".txt, .rtf, .xml" and specialized formats like ".epub" for ebooks and ".wpd" for WordPerfect documents.

## Images
Visual content is supported through ".jpg, .jpeg, .png, .gif, .bmp, .svg, .tiff, .webp" and web formats including ".htm, .html".

## Spreadsheets
Data files encompass ".xlsx, .xls, .csv, .ods" and various legacy spreadsheet formats. The system recognizes ".tsv" and database-adjacent formats like ".dbf".

## Audio
Sound files are accepted with a "20Mb" maximum file size limit. Supported formats include ".mp3, .m4a, .wav, .webm" and video audio tracks from ".mp4, .mpeg".

This comprehensive format support enables LlamaParse to process documents across business, creative, and data-oriented workflows without requiring preliminary format conversion.
